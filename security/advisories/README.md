# Security Advisories Archive

This directory contains published security advisories for the MyXenPay mobile application.

## Naming Convention

Advisories are named using the format:
```
MYXEN-SEC-YYYY-NNN.md
```

Where:
- `YYYY` = Year of disclosure
- `NNN` = Sequential number within the year
